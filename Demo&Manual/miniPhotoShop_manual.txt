3. 사용 과정
3. Manual

파일을 선택해서 이미지 파일을 불러들이면 상단에 이미지가 뜬다.
Select a Image File, then the image will be set on the upper area.

주어진 포토샵 기능의 버튼을 누르면 하단에 기능이 적용된 이미지가 뜬다.
If you click photoshop func button, The changed image will be set on the lower area.

Save Image File을 누르면 로컬에 이미지가 저장된다. (PC)
If you click the Save Image File, It will be saved in local (only available in PC)
-> 저장될 때 하단에 표시되므로 누르면 포토샵이 적용된 이미지를 확인할 수 있다.
-> When the image file is saved, It will be showed saved file on the lower area of browser,
 then you can check the changed image.

(여러 버튼을 연속해서 적용시킬 때 결과 이미지가 비정상적으로 출력될 수 있으므로
다른 기능의 버튼을 누를 때 잠깐의 시간을 두고 누르면 정상적으로 작동한다.)
(When you click several buttons repeatedly, Please click the button at regular intervals.)
